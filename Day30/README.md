# Day 30 Lab Summary

## Commands Learned

```bash
pwd
mkdir
touch
ls -r
date
uptime
```

## What I Learned

* Display the current date and time using the `date` command.
* Check system uptime with the `uptime` command.
* Determine the current working directory using `pwd`.
* Understand the difference between **Absolute Paths** and **Relative Paths**.
* Create directories using the `mkdir` command.
* Create empty files using the `touch` command.
* Verify filesystem changes using the `ls` command.
* Use `ls -r` to list files and directories in reverse order.

## Conclusion

This lab helped me understand basic Linux filesystem navigation and file management commands. It improved my understanding of creating directories/files and working with paths in Linux.
